# Multicine Testing - Playwright + Cucumber

Tests automatizados E2E para multicine.com.bo

## Requisitos

- Node.js 18+

## Instalación

```bash
git clone https://github.com/TU-USUARIO/multicine-testing.git
npm install
```

## Ejecutar Tests

```bash
npm run gherkin
```

## Control de Versiones

- Sistema: Git
- Remoto: GitHub
- CI/CD: GitHub Actions

## Ramas

- `main`: Código en producción
- `develop`: Rama de integración
- `feature/HU-XX`: Features nuevas
- `bugfix/BUG-XX`: Correción de bugs
